API INTEGRATION (13 Endpoints)
═══════════════════════════════════════════════════════════════════════════════

Authentication:
  POST /api/auth/admin-login
    → { accessToken: string }

Reports (4):
  GET  /api/admin/reports?page=1&status=OPEN
  GET  /api/admin/reports/:id
  POST /api/admin/reports/:id/review
  POST /api/admin/reports/:id/dismiss

Users (6):
  GET  /api/admin/users?page=1&search=query&role=ADMIN&status=ACTIVE
  GET  /api/admin/users/:id
  POST /api/admin/users/:id/suspend
  POST /api/admin/users/:id/unsuspend
  POST /api/admin/users/:id/role
  POST /api/admin/users/:id/reset-strikes

Analytics (1):
  GET  /api/admin/analytics?metricKind=DAU&days=30

Audit Log (1):
  GET  /api/admin/audit-log?page=1&action=USER_SUSPEND

QUICK START
═══════════════════════════════════════════════════════════════════════════════

1. Navigate to admin panel:
   cd admin-frontend

2. Install dependencies:
   npm install

3. Create environment file:
   cp .env.local.example .env.local

4. Start development server:
   npm run dev

5. Visit in browser:
   http://localhost:3001

6. Login with admin credentials

7. Start moderating!
